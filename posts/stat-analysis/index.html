Stat Analysis

2020-04-02

I wrote this using Kivy and Python for my A level computer science coursework, I chose this specific project because I felt that it would give me a good oportunity to further my understanding of statistical methods and user interface design.

While the program certainly isn’t a fully fledged statistical tool kit it due to time constraints related to the A level course it is relatively easy to add in new components. For example the application can only currently import data sets from CSV files however it wouldn’t be overly challenging to add in new ways to import data, which is something I may add in the future.

Currently the program can perform polynomial regression, K means clustering, logistic regression and can generate standard summary statistics. It can also make and export scatter plots, bar charts and graphs showing the previously mentioned statistical methods.

Future improvements that may be made could include:

  • Adding a ‘shell’ mode in which you could define probability distributions like X~Binomial(10,0.2) then you could calculatue probabilities from those distributions like P(X>3)
