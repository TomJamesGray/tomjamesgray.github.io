Calculator Project

2020-04-02

This project has gone through many iterations as I initially used Tkinter, however as I learned and grew to quite enjoy kivy I ported it to kivy. Then as I learnt about reverse polish notation I rewrote the parsing of calculations using the shunting yard algorithm to learn more about how reverse polish is used, something which was challenging but enjoyable

This calculator also has a graphing mode with support for animation through the use of variables which can be defined with limits and then set to gradually go from their minimum to their maximum, similar to that on desmos.
