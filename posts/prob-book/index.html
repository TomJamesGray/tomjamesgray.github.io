Prob Book

2020-04-02

For a long time I have been planning on making a calcualtor that allows users to easily define distributions and calculate probabilities in the classical maths notations, eg X~N(0,1) to define a standard normal distribution.

I initially started of this project using the shunting yard algorithm to parse the user input that I have used in previous projects however as the project grew in complexity I decided to switch over to lark parser which made it incredibly easy to add new features as required such as booleans. Also thanks to it’s built in Transofmer class it was very easy to evaluate the tree the parser bullt.

I also decided to implement a jupyter kernel to allow for a more intuitve and interactive user experience than the command line interface.

This project is available on Pip and can be installed with pip install prob-book
