Prob Book

2020-04-02


For a long time I have been planning on making a calcualtor that allows users to easily define distributions and calculate probabilities in the classical maths notations, eg X~N(0,1) to define a standard normal distribution. I initially started of this project using the shunting yard algorithm to parse the user input that I have used in previous projects however as the project grew in complexity I decided to switch over to lark parser which made it incredibly easy to add new features as required such as booleans.

Calculator Project

2020-04-02


This project has gone through many iterations as I initially used Tkinter, however as I learned and grew to quite enjoy kivy I ported it to kivy. Then as I learnt about reverse polish notation I rewrote the parsing of calculations using the shunting yard algorithm to learn more about how reverse polish is used, something which was challenging but enjoyable This calculator also has a graphing mode with support for animation through the use of variables which can be defined with limits and then set to gradually go from their minimum to their maximum, similar to that on desmos.

Cipher Crack

2020-04-02

I started this project while working on the Southampton university cipher challenge as I wanted an easy way of testing words my team and I thought were likely to be the passphrases, also I wanted the ability to leave this running over long periods of time in an attempt to brute force it by using a dictionary attack. The initial cipher text is decoded with a passphrase, either from a file of words (such as a dictionary) or from a predefined function, such as trying all possible words of length 6.
